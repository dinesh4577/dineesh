  • What are tags? Mention briefly of the function of a tag.

    A tag is a coded HTML command that indicated how the part of web pages should be displayed. HTML tags are the hidden keywords within a web pages which define how the web browser displays the content. All HTML tags are contained with angle brackets ( < >) eg, <Head>, <Body>, etc.

    There are two types of tags:

    1. Container Tags ( Paired Tag)
    1. Empty tag ( Singular Tag)
  • Define link. Mention the different types of link.

    Hyperlink is to link to other web page, the section of the same web pages or websites. The < A > tag is used to link web pages.

    The different types of links are:

    1. Text hyperlink
    1. Image hyperlink
    1. Bookmark hyperlink
