Computer System

  1. Define a computer system.

    The integration of input unit, processing unit, output unit along with necessary software working together to perform the certain task is known as computer system.

  1. List the different components of the computer system.

    There are five main hardware components in a computer system: Input, Processing, Storage, Output and Communication devices. Are devices used for entering data or instructions to the central processing unit.

  1. Define computer hardware with examples.

    Computer hardware is the physical components that a computer system requires to function.

  1. Define computer software with examples.

    The software is the collection of program that makes the computer work. In another word, it is the set of program that operates and control hardware and performs the specific task. Types of software are:

    1. System Software
    1. Application Software

  1. Explain the input unit in brief with examples.

    The user interacts with the computer via the I/O unit. The input units are used to provide the instruction to computer. They accept data from the user and convert the data into a form that is understandable by the computer. The input is provided to computer using input devices like keyboard, trackball, mouse, scanner, microphone, etc.

  1. What is an output unit? Give example.

    The user interacts with the computer via the I/O unit. The output unit provides the result as output in a form that is understandable by the user. Outputs are provided through the output devices like monitors and printers. Such output can also be stored in the storage devices like Hard disk, floppy disk, CD ROM, DVD ROM, flash drive (pen drive), etc.

  1. What is memory unit?

    Memory unit is a component of a computer system. It is used to store data, instructions and information. It is also known as a main/primary/internal memory.

    There are two key types of primary memory: RAM, or random access memory. ROM, or read-only memory.

  1. Define a register.

    A processor register (CPU register) is one of a small set of data holding places that are part of the computer processor.

  1. What is booting?

    The process of loading system files into computer's memory from disk is called booting.

    It starts when the computer is turned on. It makes computer ready to work. In the booting process, the command interpreter and system files are loaded into computer memory. There are two types of booting: Cold booting and warm booting

  1. What is ALU?

    An arithmetic-logic unit (ALU) is the part of a computer processor (CPU) that carries out arithmetic and logic operations on the operands in computer instruction words.
